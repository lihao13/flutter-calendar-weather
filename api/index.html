API 端点

获取天气数据：

/api/weather.php?city=城市名

示例：

查看北京天气

响应示例

{
  "success": true,
  "data": {
    "city": {
      "name": "北京",
      "lat": 39.9042,
      "lon": 116.4074
    },
    "weather": {
      "temperature": 22.5,
      "humidity": 65,
      "description": "晴朗"
    },
    "hourly": [
      {
        "time": "2026-03-25T10:00:00+08:00",
        "temperature": 22.5
      }
    ]
  }
}

  • 上海

  • 深圳
